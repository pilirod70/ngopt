Installing from source code

Installation from source code requires a c++ compiler (the makefile assumes that you have g++). Fish is known to successfully compile with g++ 3.1 and 3.2 (to see what version of g++ you have type ``g++ -v").

The code is known to compile and run on at least some versions of Solaris, RedHat Linux, Windows and MacOSX. It should compile and run on all systems with g++ installed. For windows environments, MinGW(http://www.mingw.org), provides a g++ compiler that works well. To install, type the following commands.

  1. tar -xzvf FISH-1.0.tar.z
  2. cd FISH-1.0
  3. make all
  4. cp bin/fish /usr/local/bin (may require superuser priviliges)
